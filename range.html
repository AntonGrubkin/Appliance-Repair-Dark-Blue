Common ovens and range issues and their potential fixes

Burners won't light:

Check if the burner caps and heads are clean and free from debris. Make sure the burners are properly seated on the range. If the problem persists, the issue could be with the igniter or the spark module.

Oven not heating:

Check if the oven is properly plugged in and the circuit breaker is not tripped. Make sure the oven door is fully closed. If the problem persists, the issue could be with the heating element or the temperature sensor.

Uneven heating:

Make sure the oven racks are properly positioned. Check if the oven temperature is set correctly. If the issue persists, the problem could be with the temperature sensor or the heating element.

Self-cleaning function not working:

Check if the door is fully locked and the oven is properly plugged in. Make sure the self-cleaning cycle is set correctly. If the problem persists, the issue could be with the control panel or the door lock assembly.

Oven light not working:

Check if the light bulb needs to be replaced. If the problem persists, the issue could be with the light socket or the switch.

Oven door not closing properly:

Check if there is anything obstructing the door. Make sure the hinges are properly aligned. If the problem persists, the issue could be with the door gasket or the hinge springs.

Control panel not responding:

Check if the range or oven is properly plugged in and the circuit breaker is not tripped. Make sure the control panel is not locked. If the issue persists, the problem could be with the control board or the touchpad.
